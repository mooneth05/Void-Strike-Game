⚡ VOID STRIKER

A futuristic arcade space shooter built with pure HTML5 Canvas, JavaScript, and CSS.

🚀 About

VOID STRIKER is a fast-paced space shooter where players battle waves of hostile enemies, collect powerful upgrades, unlock achievements, and compete for the highest score.

Designed with a neon cyber-space aesthetic and optimized for both desktop and mobile devices.

---

🎮 Features

- Endless wave-based gameplay
- Multiple enemy types
  - Basic Fighters
  - Fast Scouts
  - Heavy Tanks
- Power-Up System
  - Shield
  - Rapid Fire
  - Spread Shot
  - Time Slow
  - Nuke
- Achievement System
- Local Leaderboard
- Particle Effects
- Mobile Touch Controls
- High Score Saving
- Responsive Design

---

🕹 Controls

Desktop

Key| Action

⚡ Power-Ups

🛡 Shield

Restore one life.

⚡ Rapid Fire

Greatly increases fire rate for a limited time.

🔱 Spread Shot

Fire multiple projectiles simultaneously.

❄ Time Slow

Slow down enemy movement and attacks.

💥 Nuke

Destroy all enemies currently on screen.
